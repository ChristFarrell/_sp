# AI Food Assistant

A modern AI-powered meal planning application with nutritional tracking.

## Features

- 3 meal categories: Breakfast, Lunch, Dinner
- AI-generated daily meal suggestions
- Approve or regenerate meal options
- Nutritional chart analysis (protein, carbs, fats, etc.)

## Tech Stack

- React + Vite
- Tailwind CSS
- Recharts for nutrition visualization

## Getting Started

```bash
npm install
npm run dev
```
